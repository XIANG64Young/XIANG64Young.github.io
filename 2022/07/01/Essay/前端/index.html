Essay——前端总结

本文最后更新于：2022年8月1日 上午

看书、看视频涨技术没有做项目来的快 ，但是驱动学习的一定是你的目标，你的知识需要

翻到学习文件，应该还可以Github资源，主页地址在文字资源都在这

面试刷题：高级前端面试（vx公众号）、前端面试星球（vx公众号）、https://muyiy.cn/question/

基础：

  1. 环境搭建 webstorm，hbuilder，vscode（最好），sublime。
  2. html MDN HTML Doc
  3. css MDN CSS Doc
  4. JavaScript MDN JavaScript DOC

要先看基础，b站上好多视频，选择一个喜欢的老师教学视频跟着看，html ，css，js，先看基础的，看完之后，搜索高阶js，js一定要多看，html，css可以换个老师再看一遍，会有不同的收获（我是这样学的，看完一个老师的视频，有不懂的地方，我重新找一个老师，选择性看） （师姐说的）

我也认为基础是重要的，框架并没有那么重要，你需要成为一个做框架的人，而不是一个只会用框架的人。会用框架说明不了什么（但是能找到工作），我觉得前端的核心会了，框架应该也是挺顺的。
学完基础后，可以仿照电商网站（例如京东、小米）做首页的布局。我记得当时学校培训的时候，我们老师就是这么干的。

进阶：

  1. 纯js操作dom、jQuery、Ajax等。jQuery没有过时（师姐说的）。
  2. 深入学习js：知道作用域和闭包，解读原型链，原型替换，this指向和对象原型等，es6，es7的新语法。es6，这部分属于JS新增的语法，面试必问。其中，关于 promise、async 等内容要尤其关注，学习地址在下方。
  3. HTML5和CSS3。要熟悉其中的新特性。 Less 中文网
  4. ui框架之旅推荐从bootstrap开始，移动Web开发、Bootstrap等。要注意移动开发中的适配和兼容性问题。
  5. UI框架：Element UI
  6. 前端框架：vue，react，angular2选一种，我建议从vue开始，react的jsx语法不适合你入门去学习，angular2的typescript语法难度也比较高，vue学起来平滑些，结合vue，学一下vue-router，vuex，element,ui，axios，webpack。做个项目玩一下这个全家桶。
  7. 前端工程化
  8. 前端综合

学习框架，react或者vue或者angular，这三个框架看自己兴趣选择一个先学，再从b站上找一些项目，尚硅谷的之类的跟着学，学习的这些都要记笔记，因为内容很多。如果以后找工作的话，还要刷题，多看看题目就好了。如果还有时间的话，这些都学了，可以多学几个框架，vue和react都掌握了是最好。（TS的内容比ES还要多。。。）

其他：

  1. git与nodejs Guides | Node.js git是基本操作，仓库这个东西还是要会的。node.js我记得vue里面就用到了一点
  2. canvas。面试时，有的公司不一定会问canvas，靠运气。如果时间不够，这部分的内容可以先不学。但如果你会，绝对属于加分项。(我记得是用来做网页小游戏的吧，我好想写过) 代码

基础项目：

  1. animate 代码 效果
  2. CSS-Inspiration代码 效果
  3. vanillawebprojects 表单验证、音视频播放器、打字等小应用 代码
  4. Ant Design 基于 Ant Design 设计体系的 React UI 组件库 React 代码 Vue代码
  5. Echarts 基于 JavaScript 的开源可视化图表库 代码
  6. ECMAScript是JavaScript的语言标准，描述了JS的能力，API，基本实现步骤。 es6地址

中级项目：

  1. particles.js 当我们浏览网页的时候，会看见一些网站的背景有粒子效果，看完是不是也很好奇它是怎么实现的呢？在particles.js这个库里面，我们可以制造许多粒子效果，用在网页的背景可是相当好看的。演示 代码
  2. reveal.js是一款网页ppt，界面优美，又支持许多ppt功能，同时也有许多插件满足前端的需求，例如markdown、code等代码高亮插件。演示 代码
  3. TS 现在的前端开发中，对于TypeScript的使用越来越多，尤其是大型迭代项目中，JavaScript不能满足需要，学习TypeScript会是一个必备的技能 书
  4. 音乐器 还在因为云音乐因为版权下架了一批你自己喜欢的歌曲，但是又不想单独为了几首歌换平台而烦恼吗，那你可以试试这个库，可以使用 QQ / 虾米 / 百度 / 酷狗 / 酷我 / 咪咕 / JOOX 等音源把云音乐灰色歌曲解锁播放。代码

Vue.js项目：

  1. vue-element-admin 是一个后台前端解决方案，它基于和 element-ui 实现 效果、代码 、介绍、模版
  2. 基于Vue.js + Element UI 的后台管理系统解决方案 效果、代码
  3. Vue + TypeScript + Element-Ui 支持 markdown 渲染的博客前台展示
    这个博客主挺6的，vue、react的项目 啥都做了 他给的其他项目连接都在这个网页的下面

  3. 网易云音乐第三方代码
  4. 饿了么，还原度相当高，实现了各类动效代码
