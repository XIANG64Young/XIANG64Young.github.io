Toolkit——git相关操作

本文最后更新于：2022年8月1日 上午

git拉取代码

\1) 初始化：git init

主要用来初始化一个空的git本地仓库。执行完上面的命令，当前目录下会自动生成.git隐藏文件夹，该隐藏文件夹就是git版本库

\2) 将本地仓库与远端仓库建立一个链接，并命名远端仓库为origin

git remote add origin master url

本地仓库默认为master分支，url为拉取的仓库地址

\3) git pull origin master

拉取origin远端仓库到本地master分支

git拉取代码完成，下面介绍如何用git命令提交代码

git提交代码

1） 在gitee官网新建仓库

2）对该仓库进行初始化

得到如下图：

3）在本地拉取该仓库，通过如下命令行实现：

git clone url 其中url为仓库地址

4） 在本地拉取的仓库文件夹再复制您需要上传的代码

5） 将上传的代码添加到暂存区 git add .

\6) 将暂存区的内容添加到本地仓库 git commit -m "第一次提交"

\7) 将本地仓库推送到远端仓库git push origin master

拉取代码和提交代码大致是这个步骤，git可以在算法开发中大大提高开发效率。��**
