python程序在命令行执行提示ModuleNotFoundError: No module named ‘XXX’ 解决方法

在模块中添加

interrupted by signal 9: SIGKILL

终端top查看cpu内存未满，nvidia-smi查看gpu也未占满。
可能原因就是：pycharm配置运行内存太小。

  1. 找到pycharm的安装目录下的 pycharm.vmoptions
  2. 右键用文本编辑器打开，修改前两项pycharm 的最小、最大内存：Xms和Xmx

也有可能是你调用的变量太大，导致内存过载，建议使用批量处理方式。

python中出现IndentationError:unindent does not match any outer indentation level错误

1.对于此错误，最常见的原因是，的确没有缩进。根据错误提示的行数，去代码中看了下，看起来没有什么问题呀，都有缩进，而且语法也没有错误呀。

2.仔细研究了下代码，发现真的看不出什么问题，突然想到了，把当前python脚本的所有字符（包括空格和tab字符）都显示出来看看到底有没有缩进或者是其他特殊的字符。

最简单的解决办法，格式化一个代码格式。如果是在pycharm内的话Ctrl+Alt+L

RuntimeError: unsupported operation: some elements of the input tensor and the written-to tensor refer to a single memory location. Please clone() the tensor before performing the operation.
